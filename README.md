Introduction
============
This is an App to add watermarks to your images. Build with [Next.js](https://nextjs.org/), [React](https://reactjs.org/), [TailwindCSS](https://tailwindcss.com/)

Getting Started
---------------
1. Clone this repository
```bash
git clone <url>
```
2. Install dependencies
```bash
npm install
```
3. Run the app
```bash
npm run dev
```
